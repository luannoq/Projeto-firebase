## 🎯 Objetivo do Projeto
Desenvolver uma aplicação mobile funcional que permita ao usuário gerenciar suas notas pessoais com segurança, utilizando serviços de autenticação e banco de dados em nuvem.

## 🚀 Funcionalidades
### Autenticação (Firebase Auth)
- [x] Cadastro de novos usuários (E-mail e Senha).
- [x] Login de usuários cadastrados.
- [x] Persistência de estado (proteção de rotas).
- [x] Logout seguro.

### Gerenciamento de Notas (Cloud Firestore)
- [x] **Criar:** Adicionar novas notas à base de dados.
- [x] **Listar:** Visualização das notas vinculadas exclusivamente ao usuário logado.
- [x] **Editar:** Atualizar o conteúdo ou título de notas existentes.
- [x] **Deletar:** Remover notas de forma permanente.

## 🛠️ Tecnologias Utilizadas
- **React Native**
- **Firebase Authentication** (Gestão de usuários)
- **Cloud Firestore** (Banco de dados NoSQL)
- **React Navigation** (Navegação entre telas)

## 📋 Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
- Node.js
- Expo CLI
- Um emulador Android/iOS ou o app Expo Go instalado no celular.

## 🔧 Como rodar o projeto
1. Clone este repositório:
       git clone https://github.com/luannoq/Projeto-firebase.git

2. Instale as dependências:
       npm install

3. Configure o seu arquivo de credenciais do Firebase (normalmente firebaseConfig.js) com as chaves do seu projeto no console do Firebase.

4. Inicie o projeto:
       npx expo start
